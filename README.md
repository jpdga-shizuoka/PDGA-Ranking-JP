# 日本在住PDGA登録選手の年間PDGAランキング作成ツール

本ツールは、
日本在住のPDGA登録選手を対象にPDGAが公開する年間ポイント並びに総獲得賞金(プロのみ)によって順位付けをする機能を提供します。

本ツールの概要は以下の通りです:

1. 現行のPDGA有効会員が対象です。未会員または失効会員は対象外です。
1. 国内のPDGA公認大会だけでなく国外のPDGA公認大会の実績も本集計に含まれます。
1. 順位付けは、カテゴリー別の区分を設けず、プロとアマのニ区分で行います。
1. プロ登録選手は、PDGAに公開された`Pro Totals`を集計します(プロ登録選手図参照)。
1. アマ登録選手は、PDGAに公開された`Pro Totals`と`Am Totals`の合計を集計します(アマ登録選手図参照)。
1. シーズン途中でアマからプロへ登録変更した選手は、本ツールの都合上、プロ登録選手として扱われ、アマのポイントは加算されません。

## 使い方

本ツールでは、以下の手順でランキングを作成します:

1. 日本在住者のPDGA有効会員リストの作成する
1. 同リストに基づいて、各登録選手の実績を集計する
1. 集計をもとにランキングを作成する

項1のリスト作成は、年を跨ぐと有効会員が失効会員になる可能性があるため、可能な限り年内に作成することを推奨します。
また項2の実績の集計は、年末の大会実績を確実に反映させるため、翌年の1月もしくは2月の第1火曜日以降に実施することを推奨します。
